Here are the launch codes:
In your Linux terminal, Navigate the folder you want to download this to then type these codes once: 
* git clone https://github.com/DamilolaOwolabi/DS_7346.git
* virtualenv env
* source env/bin/activate
* pip install -r requirements.txt
* python manage.py runserver

The last output would shouw you a link which you can load to your browser.

  The website should be running now. Type 'Ctrl + C' to end the run. To reload the link you just need to type these 2 codes:
  * source env/bin/activate
  * python manage.py runserver
